# Supply-chain-management
A small database system designed to manage products, suppliers, orders, and inventory using ER modeling, normalized tables, and SQL queries. It helps organize and track essential supply chain information efficiently.

## Tech Stack

- **Frontend:** React, JavaScript, HTML, CSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Auth:** JWT-based authentication
